Open-Meteo API

장점:
 완전 무료
 API 키 불필요 (회원가입 없음!)
 CORS 지원 (브라우저에서 바로 사용 가능)

날씨 API URL 형식:
https://api.open-meteo.com/v1/forecast
  ?latitude=37.57       ← 위도
  &longitude=126.98     ← 경도
  &current_weather=true ← 현재 날씨 포함

도시 검색 API (Geocoding):
https://geocoding-api.open-meteo.com/v1/search
  ?name=Seoul           ← 도시 이름
  &count=1              ← 결과 개수
  &language=ko          ← 한국어
